# Product Scope

## Amaç
Kullanıcıdan alınan yapılandırılmış veriler ile anlamlı ve duygusal olarak tutarlı bir hayat hikayesi üretmek.

## Input
- Ad
- Yaş
- Doğum yeri
- Aile bilgileri
- Eğitim
- Kariyer
- Önemli olaylar
- Dönüm noktaları
- Ton (ilham verici, dramatik, sade, vb.)

## Output
Bölümlenmiş hikaye:
- Giriş
- Çocukluk
- Eğitim
- Kariyer
- Dönüm noktaları
- Bugün
- Gelecek vizyonu

## Core Value
Ham bilgiyi anlamlı bir anlatıya dönüştürmek
